# Project Overview

## Description

Financial Metrics Dashboard — a full-stack application for visualizing financial KPIs (Key Performance Indicators). The dashboard displays income, outcome, profit, and profit margin metrics through an interactive UI with line charts.

## Purpose

Educational project by **4Geeks Academy** for the **AI Engineering** bootcamp. Designed to be worked on by AI agents who inspect both frontend and backend, document rules, and refine the project workflow.
